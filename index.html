Temperature Conversion

This ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 


def c_to_f(n)
	return (n * 9 / 5) + 32
end

puts "Enter degrees in Celcius:" 
celcius = gets.chomp
celcius_float = celcius.to_f
cel_to_fah = c_to_f(celcius_float)

puts "The temperature is #{cel_to_fah} degrees Fahrenheit"

def get_ordinal_1_digit(n)
  right_most_digit = n % 10
       
  if right_most_digit == 0
    return n.to_s + "th"
  elsif right_most_digit == 1
    return n.to_s + "st"
  elsif right_most_digit == 2
    return n.to_s + "nd"
  elsif right_most_digit == 3
    return n.to_s + "rd"
  else
    return n.to_s + "th"
  end
end

Foobar

This ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 


def foobar(n)
  count = 1
  
  while count <= n
    divisible_by_3 = count % 3
    divisible_by_5 = count % 5
  
    if divisible_by_3 == 0 && divisible_by_5 != 0
      puts "Foo"
    elsif divisible_by_5 == 0 && divisible_by_3 != 0
      puts "Bar"
    elsif divisible_by_3 == 0 && divisible_by_5 == 0
      puts "FooBar"
    else
      puts "#{count}"
    end #end if
    
    count = count + 1
    
  end #end while
end #end def

puts "How many items do you want to see?"
items = gets.chomp.to_i

foobar(items)
